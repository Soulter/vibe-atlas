# Vibe Atlas

Vibe Atlas is a canvas-based workspace for running and organizing multiple terminals, notes, and parallel coding sessions.

<img width="600" height="1800" alt="image" src="https://github.com/user-attachments/assets/86bc5e90-2d4c-489c-a1aa-c2d9f7a7440f" />

## What it does

- Place multiple terminal windows on a shared canvas
- Save and reopen workspace layouts
- Add free-form notes anywhere on the canvas
- Pan and zoom like a visual workspace, not a tab list
- Track terminal status: `idle`, `running`, `exited`

## Shortcuts

- `Cmd/Ctrl + T`: create a new terminal
- `Cmd/Ctrl + S`: save the current workspace
- `Shift + drag`: pan the canvas
- `Double click`: create a note

## Development

```bash
npm install
npm start
```
